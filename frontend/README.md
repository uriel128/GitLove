# GitLove Frontend

## Local run

```bash
cd frontend
cp .env.local.example .env.local
npm install
npm run dev
```

Default local URL:
- `http://localhost:3000`

Expected backend:
- `http://localhost:4000/api`

Optional Supabase frontend vars:
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
